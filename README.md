## About The Project

My simpler implementation of Ethereum Name Service (ENS)

## Built With

- NextJS
- Hardhat
- TailwindCSS

## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/yektas/ens.git
   ```
2. Install packages

   ```sh
   yarn install
   ```

3. Run the application

   ```sh
   yarn dev
   ```
